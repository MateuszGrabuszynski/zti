Extracting and serializing data from client.

server.serializer.extract_address(data)¶

Finds all <…>, chooses the last one and removes the unnecessary brackets

Parameters:data – Data sent by the client.
Returns:Just the extracted address from triangle brackets.
server.serializer.extract_begin_end(data)¶

Finds nif:beginIndex and nif:endIndex values.

Parameters:data – Data sent by the client.
Returns:Begin index and end index, -1 if error.
server.serializer.extract_string(data, begin, end)¶

Extracts the main string from the message sent by the client and cuts it specifically.

Parameters:
  • end – The end of the string.
  • begin – The beginning of the string.
  • data – Data sent by the client.
Returns:

String part of the message.
server.serializer.prepare_response(series, data, address)¶

Prepares response for the client.

Parameters:
  • series – Graphed series.
  • data – Original data from the client.
  • address – Address given in <> brackets by the client.
Returns:

Response to be sent back to the client.
server.serializer.remove_signs(word)¶

Removes dots, commas etc. from beginnings and endings. ‘Inc’ gets dot at the end and is returned as ‘Inc.’

Parameters:word – Just a word from the sentence.
Returns:Word without dots and other signs if they are on the beginning or the end of it.

server.server.handle_client(conn)¶

Main client handling function. Runs in a new thread while the client is connected to the server. Closes the connection when no data is received.

Parameters:conn – Connection parameters with a connection hook.
server.server.handle_data(data)¶

Allows to handle data sent by the client.

Parameters:data – Raw data sent by the client.
Returns:Response that may be sent back to the client.
server.server.run_server(host='', port=8080, max_conn=10)¶

Allows to run a server

Parameters:
  • host – Allows to run the server only locally (e.g. 127.0.0.1), empty runs everywhere (like 0.0.0.0).
  • port – Port on which the server runs.
  • max_conn – Maximum allowed number of simultaneous incoming connections.
